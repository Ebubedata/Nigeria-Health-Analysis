# Nigeria-Health-Analysis
### Description 
This presentation analyzes health data across 37 Nigerian states using an interactive dashboard 

### Objective Questions
- Which States has the highest incidence of Malaria and Cholera?
- Which States has the highest death rate due to Cholera?
- Which group do we have the highest diesease occurrence,  rural or urban?
- Which diseases has the highest death rate in 2010 ans 2018?
- Which year do we have the highest incidence of Meningitis?
- Which Age group are more perceptible to Diarrhoea?
  ### Data Analysis Process
- Data Cleaning - Removed errors and duplicates, changed data types, Renamed Columns, Replaced values
- Created Age Group column - Used conditional column
- Analysis with Charts
- Created Slicers for filtering.
- Calculated KPIs
### Insights
- Abia State recorded the highest death rate due to Cholera.
- Urban settlements recorded more disease cases than rural settlements.
- Top 3 states with the highest combined cases are Kebbi, Niger and Kaduna.
- Cholera had the highest death rate in 2010.
- Ebola recorded the highest death rate in 2018.
- Highest incidence of Meningitis occurred in 2015.
- Lowest incidence of Meningitis occurred in 2010.
- Adults are the most susceptible group to diarrhoea. Teenagers are the second most affected group.
### Recommendations 
- Strengthen cholera response in Abia State
- Increase disease prevention in urban settlements
- Prioritize Kebbi, Niger, and Kaduna for intervention
- Monitor meningitis outbreak trends annually
- Implement adult-focused hygiene education programs
